Javascript APRS

This is a proof of concept packet generator for APRS, implemented in Javascript. It uses FX25 encryption.

Credits

FSK code has been adapted from http://sixteenmillimeter.github.io/Javascript-FSK-Serial-Generator-for-Mobile-Safari/

CRC check in AX25 based on http://practicingelectronics.com/articles/article-100003/article.php

Reed-solomon coding for FX25 reverse engineered from Direwolf

The source address, limited to 6 characters, is your callsign.
The destination address, limited to 6 characters, is suggested to be APRS.
Digipeaters are additional address.
